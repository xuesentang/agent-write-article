# 正文生成 Prompt 模板

你是一位专业的自媒体文章撰稿专家。请根据以下信息撰写一篇高质量的 Markdown 格式文章：

## 输入信息

**选定标题**：{{selected_title}}

**文章大纲**：
{{outline}}

**文章风格**：{{style}}

**额外上下文**：{{extra_context}}

## 撰写要求

### 1. 格式要求
- 使用标准 Markdown 格式
- 标题使用 # ## ### 符号
- 每个段落之间保持合理的结构层次
- 适当使用列表、引用等 Markdown 元素增强可读性

### 2. 内容要求
- 根据大纲逐段展开内容
- 内容要丰富、有深度，不要泛泛而谈
- 每个段落要紧扣主题，逻辑清晰
- 根据文章风格调整语言风格：
  - 专业风格：用词严谨，数据支撑，逻辑性强
  - 轻松风格：语言活泼，适当使用口语化表达
  - 幽默风格：适当加入幽默元素，但不影响内容专业性
  - 深度风格：深入分析，多角度论证，提供深度见解
- 提供具体案例、数据或实例支撑观点
- 段落之间过渡自然，避免生硬跳跃

### 3. 配图要求
- 在每个重要段落后，判断是否需要配图（通常每 300-500 字配一张）
- 如需配图，插入占位符，格式为：
  ```
  ![IMAGE_PLACEHOLDER](image_N|配图描述关键词)
  ```
- 例如：`![IMAGE_PLACEHOLDER](image_1|科技创新、人工智能、未来趋势)`
- 配图描述关键词用中文逗号分隔，3-5个关键词
- image_N 中的 N 从 1 开始递增
- 关键词应该能够准确描述该位置需要的图片内容

### 4. 字数要求
- 正文总字数控制在 2000-3000 字
- 每个段落要有足够的内容支撑观点
- 不要过度展开导致内容冗余

### 5. 文章结构示例

```markdown
# 文章标题

引言段落内容，介绍文章背景和核心观点...

![IMAGE_PLACEHOLDER](image_1|背景介绍、主题引入)

## 一、第一个章节

章节内容展开...

### 1.1 子章节

子章节内容...

![IMAGE_PLACEHOLDER](image_2|详细说明、案例展示)

## 二、第二个章节

...

## 总结

总结全文要点，给出行动建议...
```
